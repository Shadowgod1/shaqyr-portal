# PrivateEvents 🎊

Современный веб-портал для организации частных мероприятий (свадьбы, дни рождения, вечеринки) с доступом по секретному коду.

## 🚀 Стек технологий
- **Frontend**: React + Vite
- **Styling**: Vanilla CSS (Modern CSS Variables + Glassmorphism)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Backend**: Supabase (PostgreSQL + Auth)
- **Routing**: React Router DOM

## ✨ Основные функции
- **Приватность**: Доступ к деталям события только по уникальному коду.
- **Темы**: Автоматическая смена дизайна под тип события (Свадьба, ДР, Вечеринка).
- **RSVP**: Система подтверждения участия гостей.
- **Dashboard**: Личный кабинет организатора для управления событиями.
- **Интерактив**: Стена пожеланий и галерея моментов.

## 📦 Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone <your-repo-url>
```

2. Установите зависимости:
```bash
npm install
```

3. (Опционально) Настройте Supabase:
Создайте файл `.env` в корне проекта и добавьте ваши ключи:
```env
VITE_SUPABASE_URL=ваш_url
VITE_SUPABASE_ANON_KEY=ваш_ключ
```

4. Запустите проект:
```bash
npm run dev
```

## 📂 Структура проекта
- `src/components`: Переиспользуемые UI компоненты.
- `src/pages`: Основные страницы приложения.
- `src/context`: Управление состоянием (Auth, Events).
- `src/styles`: Глобальные стили и дизайн-система.
- `src/lib`: Конфигурация внешних библиотек (Supabase).
